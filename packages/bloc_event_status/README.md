Track the status of events in a bloc without updating the state.

## Installation

Add the package to your `pubspec.yaml`:

```yaml
dependencies:
  bloc_event_status: ^1.0.0
```

and then run:

```bash
flutter pub get
```

Or just install it with flutter cli:

```bash
flutter pub add bloc_event_status
```
